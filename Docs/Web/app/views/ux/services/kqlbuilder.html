El servicio KQLBuilder es un servicio de apoyo para crear URL's con el formato KQL (Karma Query Language), sobre un recurso Web con tecnología basada en Karma.

Recomendaciones:

  • Para casos de consulta hacia un recurso Web basado en Karma , es recomendable utilizar el método kql que se encuentra en el servicio $Api

Uso Basico

app.controller('MyController', function($scope, KQLBuilder) { var url = "http://demoApp.valentys.com/API/v1/Account"; var kql = { limit: 1, offset: 2, orderBy: 'firstname asc', filters: [ { name: 'email', operator: 'contains', value: 'dmunoz@valentys.com' } ] }; console.log( KQLBuilder.build(kql) ); //http://demoApp.valentys.com/API/v1/Account?$limit=1&offset=2&filter=email contains 'dmunoz@valentys.com'&orderBy=firstname asc });

Métodos

  • KQLBuilder.build(endpoint, kql);

    Compila la url con la configuración en formato kql y retorna la URL completa.

    Nombre Tipo Descripción
    endpoint String

    URL base del recurso Web con tecnología basada en Karma.

    kql JSON

    Objeto con la configuración a establecer en el recursos Web con tecnología basada en Karma.

    Retorno Descripción
    String

    Retorna la URL compilada con el formato JSON.
