Atributos

  • Nombre Tipo Descripción
    title String

    Tituló a establecer en atributo document.title
    Si se establece un titulo, se ejecutará además el evento gale-page:title:changed

    on-keydown Function

    Función que se ejecuta cuando el usuario presiona una tecla sobre el elemento gale-page
